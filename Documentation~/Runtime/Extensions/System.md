# Runtime/Extensions/System

## Mục tiêu
Helper cho các kiểu system như string, enum, random, timespan, reflection nhẹ...

## Scripts
- `DateTimeExtensions.cs`
- `EnumExtensions.cs`
- `GuidExtensions.cs`
- `PlayerPrefsExtensions.cs`
- `StringBuilderExtensions.cs`
- `StringExtensions.cs`

## Ví dụ
- String helpers: IsNullOrWhiteSpace, TrimSafe...
- Enum helpers: ToInt, HasFlagSafe...

> Nếu bạn muốn, mình có thể tự động trích danh sách public API từ từng file để đưa vào docs (nâng cấp thêm).
